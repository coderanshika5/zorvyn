# Finance Dashboard

## Overview
This project is a simple finance dashboard built using React and Vite.  
It allows users to view financial summaries, explore transactions, and understand spending patterns.

## Tech Stack
- React
- Vite
- JavaScript
- Plain CSS

## Features
- Financial summary cards (Balance, Income, Expenses)
- Transaction list with filtering and search
- Role based UI (Admin / Viewer)
- Spending insights

## Setup
1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`
